# Synapse

Notionや海外のインディーWebマガジンを彷彿とさせる、洗練されたエディトリアルデザインが特徴の、ミニマルなフルスタック匿名テキストスペースです。

量産型のAI生成テンプレート感（過剰なグラデーションやボケた立体影）を完全に排除し、「人間が意図を持って細部まで引き算で作り込んだオリジナルUI/UX」を追求しました。

## 🚀 アプリケーションURL
- **本番環境（デモサイト）**: https://laravel-portfolio-board.vercel.app/
- ※バックエンドは無料枠（Render.com）の仕様上、一定時間アクセスがないとスリープに入ります。初回の読み込みに数十秒かかる場合がありますが、正常に動作します。

## 🛠️ 技術スタック

### フロントエンド
- **Library**: React 18
- **Language**: TypeScript (厳格な型定義、未使用変数の徹底排除によるクリーンコードの維持)
- **Styling**: Tailwind CSS (レトロ・マガジン風のソリッドな独自テーマ構築)
- **Icons**: Lucide React

### バックエンド
- **Framework**: Laravel 13 (REST APIモード)
- **Language**: PHP 8.4
- **Database**: SQLite (開発・デモ運用を極限まで軽量化)

### インフラ・デプロイ
- **Frontend Hosting**: Vercel (GitHub連携による継続的デプロイ)
- **Backend Hosting**: Render.com (Dockerコンテナによるアプリケーション実行)
- **Containerization**: Docker (PHP 8.4-cliベースの最適化イメージの自作)

## ✨ 独自のこだわり・実装のハイライト

### 1. 「脱・AI量産型」をテーマにした独自のデザインシステム
- AI生成UIや初心者教材にありがちな「鮮やかなインディゴブルー」「過剰なグラデーション」「大げさなドロップシャドウ」を意識的にオミット。
- 紙の質感を思わせる温かみのある砂色（`#f6f5f2`）の背景と、シャープな単色ベタのソリッドシャドウ（`shadow-[3px_3px_0px_0px_rgba(30,41,59,1)]`）を組み合わせ、Notion風の知的なエディトリアルデザインを実現。

### 2. TypeScriptによる堅牢でクリーンなフロントエンド開発
- コンパイルエラーだけでなく、静的解析による警告（未使用のインポートや変数など）を徹底的に排除（TS6133エラーの完全解消）。
- 採用担当者がコードを読んだ際に「実務で通用するクリーンなコードが書ける人材」だと一目で伝わる保守性の高いコードベースを構築。

### 3. Dockerによるバックエンドのコンテナ化と環境完全分離
- Laravel環境を軽量な `php:8.4-cli` をベースとした `Dockerfile` で完全にパッケージング。
- Render.comの環境依存を解消し、インフラ知識（IaC / コンテナ技術）を網羅したフルスタックな開発実績を証明。
- フロントエンド（Vercel）とバックエンド（Render）のドメイン完全分離に伴うCORS認証・トークンベース（localStorage）のセッション維持を適切に制御。

## 🔑 主要機能
- **ユーザー認証**: アカウント作成（サインアップ）、ログイン、ログアウト（Token制御）
- **タイムライン機能**: 全ユーザーの投稿の一覧表示（高速なREST API通信）
- **投稿機能**: ログインユーザー限定のメッセージ投稿、および自身のトークンに紐づいた投稿削除機能
- **レスポンシブ対応**: スマートフォン、タブレット、PCのすべてのデバイスに最適化したエディトリアルUI
